Course Calendar Generator

This will generate a Word document with a table that prefills the dates based on the days you want. If you want more than one calendar (like a MW and a TR), refresh the page before you do the second one or else you will get two tables joined together. I don't know how to fix that yet.

This has not been tested (and never will be) on Internet Explorer or Edge.

One more thing: If you try to "Save As" in Word, it will default to "htm/html" as the file type. You will need to manually select "docx" to save it to the new format.
